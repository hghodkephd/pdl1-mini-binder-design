# De Novo Mini Binder Design Against PD-L1

Computational design of de novo mini protein binders targeting PD-L1,
using the RFdiffusion → ProteinMPNN → AlphaFold pipeline.

## Project Structure
- `notebooks/` — Jupyter notebooks for each stage
  - `01_target_preparation.ipynb`
  - `02_rfdiffusion_binder_generation.ipynb`
  - `03_proteinmpnn_sequence_design.ipynb`
  - `04_alphafold_validation.ipynb`
  - `05_analysis_and_filtering.ipynb`
- `data/` — Input structures, sequences, and results
- `figures/` — Publication-quality figures
- `scripts/` — Reusable utility functions
